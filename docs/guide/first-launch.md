# First Launch

After enabling the plugin:

- A **wave icon** appears in the left ribbon — click it to open the Cortex chat panel
- Or use the Command Palette (`Ctrl+P` / `Cmd+P`): **Cortex: Open agent panel**

::: tip Claude not found?
If Cortex can't find your Claude Code installation, a setup panel will appear. Either follow the on-screen steps to install Claude Code, or enter the full path to your `claude` binary in **Settings → Cortex**.
:::

Once the panel opens, type a message and press **Enter**. Cortex injects your vault structure and context file at the start of each session, so Claude is never starting blind.

**Try these to get a feel for it:**

- *"Summarize the note [[Home]]"*
- *"What notes do I have tagged #meeting from last month?"*
- *"Create a new note in 06_Spaces called 'Q2 Goals' with a bullet outline"*
